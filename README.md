💼 Portfólio Pessoal – Avaliação de Desenvolvimento Web

Este repositório apresenta um site de portfólio pessoal desenvolvido como parte de uma avaliação prática na disciplina de Desenvolvimento Web.

O projeto tem como objetivo demonstrar domínio de estruturação HTML, organização de conteúdo, semântica e boas práticas na construção de páginas web estáticas.

📌 Estrutura do Projeto

O site é composto por três páginas principais:

Sobre mim (index.html)
Apresentação pessoal, formação e áreas de interesse.
Projetos (project.html)
Exibição de projetos desenvolvidos, com foco em lógica de programação e aplicação prática.
Contato (contact.html)
Informações de contato e formulário para comunicação.

🛠️ Tecnologias Utilizadas
HTML5
CSS3

🎯 Objetivos do Projeto
Aplicar conceitos de HTML semântico
Estruturar páginas de forma organizada e reutilizável
Criar uma navegação consistente entre múltiplas páginas
Apresentar informações de forma clara e objetiva
Utilizar boas práticas de acessibilidade e validação de código

📎 Funcionalidades
Navegação entre páginas
Layout estruturado em seções e cards
Links externos para redes sociais
Formulário de contato
Organização visual consistente
